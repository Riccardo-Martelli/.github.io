The landing page acts as an index for these specific engineering projects:

* **[Sentiment Prediction](https://github.com/Riccardo-Martelli/Sentiment_Prediction):** An attention-based LSTM-CNN network architecture designed to scrape Google RSS/Reddit feeds and evaluate text sentiment against stock return metrics.
